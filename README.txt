UA/ENG

Проект створено з використанням blueprints

Максимальна кількість кубів: 50

В головному меню можна обрати режим, в якому відмічені куби можуть відмічати не відмічені (за виключеням очисників)

HUD відображає загальну кількість невідмічених кубів та активних очисників

Очки за куби:
	-10 за звичайний
	-100 за очисника
(В режимі де куби можуть відмічати інші куби, очки зменшені)

Керування:
	WASD - пересування
	space bar - пауза

Приклад геймплею: https://drive.google.com/drive/folders/1lEmqt2a7Slz2zX3xSS_0enk9PNwyiO1u?usp=drive_link

The project was created using blueprints

Maximum number of cubes: 50

In the main menu, you can choose a mode in which marked cubes can mark to unmarked ones (excluding cleaners)

The HUD displays the total number of unmarked cubes and active cleaners

Points for cubes:
	-10 for regular
	-100 for the cleaner
(In the mode where cubes can mark other cubes, points are reduced)

Controls:
	WASD - movement
	space bar - pause

Gameplay example: https://drive.google.com/drive/folders/1lEmqt2a7Slz2zX3xSS_0enk9PNwyiO1u?usp=drive_link
